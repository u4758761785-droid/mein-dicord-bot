1. Allgemeine Informationen

Diese Datenschutzerklärung erklärt, welche personenbezogenen Daten durch den Discord Bot „BOT_NAME“ verarbeitet werden und zu welchem Zweck dies geschieht.

Der Bot wird über die Plattform Discord betrieben. Die Nutzung des Bots setzt die Zustimmung zu dieser Privacy Policy voraus.

2. Welche Daten werden verarbeitet?

Der Bot kann folgende Daten verarbeiten:

Discord User IDs

Discord Server (Guild) IDs

Discord Channel IDs

Nachrichteninhalte nur, wenn diese für eine Funktion des Bots erforderlich sind

Es werden keine sensiblen personenbezogenen Daten (z. B. Klarnamen, Adressen) erhoben.

3. Zweck der Datenverarbeitung

Die Verarbeitung der Daten erfolgt ausschließlich zur Bereitstellung der Bot-Funktionen, zum Beispiel:

Ausführung von Befehlen (Commands)

Speicherung von Einstellungen

Moderations- oder Verwaltungsfunktionen

Fehleranalyse und Stabilität des Bots

4. Speicherung und Löschung

Gespeicherte Daten werden nur so lange aufbewahrt, wie sie für den Betrieb des Bots notwendig sind.
Nicht mehr benötigte Daten werden gelöscht.

5. Weitergabe von Daten

Es erfolgt keine Weitergabe der Daten an Dritte.

Discord selbst verarbeitet Daten gemäß der eigenen Datenschutzerklärung:
https://discord.com/privacy
